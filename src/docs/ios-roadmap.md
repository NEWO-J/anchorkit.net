# iOS Roadmap

AnchorKit for iOS is in planning. This page tracks progress toward a native Swift SDK.

## Status

| Platform | Status |
|----------|--------|
| Android SDK | Stable (v1.0.1) |
| iOS SDK | Planned |

## Planned Milestones

| Milestone | Status |
|-----------|--------|
| API design & protocol alignment with Android SDK | Planned |
| Swift SDK — core capture pipeline | Planned |
| Secure Enclave attestation integration | Planned |
| AVFoundation / camera pipeline | Planned |
| Public beta | Planned |

## Notes

- iOS SDK will target iOS 16+ and use the Secure Enclave for hardware attestation
- Planned feature parity with the Android SDK: `captureAndSubmit`, `downloadProof`, `verifyHash`
- Updates will be posted here as milestones are reached
